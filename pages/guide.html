Gestión de Productos

▼

➕ Agregar un producto

  1. Vaya a la sección "Productos" en el menú principal
  2. Haga clic en "+ Nuevo Producto"
  3. Complete los campos: nombre, descripción, precio base en USD y stock
  4. Opcionalmente, puede subir una imagen del producto (JPG, PNG, GIF - máx 2MB)
  5. Haga clic en "Guardar Producto"
💡 Tip importante: El precio base siempre debe estar en USD. Luego al vender, se aplicarán los recargos según el método de pago seleccionado.

✏️ Editar o eliminar un producto

  • En la lista de productos, use los botones "Editar" o "Eliminar"
  • Al eliminar, se le pedirá confirmación para evitar errores
  • Los productos con stock bajo (menos de 5 unidades) mostrarán una alerta en el dashboard
👥

Gestión de Gestores de Venta

▼

➕ Agregar un gestor

  1. Vaya a la sección "Gestores"
  2. Haga clic en "+ Nuevo Gestor"
  3. Ingrese nombre, apellidos y número de móvil
  4. El código se genera automáticamente (ej: GR4 para Guillermo Ramos con teléfono 4...)
  5. Si hay duplicados, se añade un número secuencial (GR4-1, GR4-2...)

💰 Comisiones de gestores

  • Cada gestor recibe el 5% del precio base del producto en CUP
  • Al registrar una venta con el código del gestor, la comisión se guarda automáticamente
  • En la sección Gestores puede ver: total generado, pendiente y pagado
  • Use el botón "Pagar" para liquidar las comisiones pendientes
⚠️ Importante: El código del gestor debe ingresarse EXACTAMENTE como aparece (mayúsculas, sin espacios) al registrar la venta.
💰

Registro de Ventas

▼

📝 Cómo registrar una venta

  1. Vaya a la sección "Ventas"
  2. Seleccione un producto del catálogo
  3. Ingrese la cantidad a vender
  4. Seleccione el método de pago (USD, CUP, MLC, PayPal, Zelle, TropiPay)
  5. Opcionalmente, ingrese el código del gestor que realizó la venta
  6. Revise el desglose de precios (recargos aplicados automáticamente)
  7. Haga clic en "Registrar Venta"

Configuración del Sistema

▼

💱 Tipo de Cambio USD → CUP

  • Vaya a "Tipo de Cambio" en el menú
  • Ingrese el valor actual (ej: 1 USD = 320 CUP)
  • El cambio se aplica automáticamente a todas las ventas futuras en CUP
  • El historial de cambios se guarda para referencia

💳 Porcentajes por método de pago

  • Vaya a "Métodos de Pago" en el menú
  • Modifique los porcentajes de recargo según sus necesidades
  • Los cambios afectan SOLO a ventas futuras
  • Puede restablecer a valores por defecto en cualquier momento

Copias de Seguridad

▼

📤 Exportar copia de seguridad

Use el botón "Importar Copia" (icono 📥) para restaurar datos desde un archivo JSON previamente exportado.

⚠️ Advertencia: Al importar una copia, se SOBRESCRIBIRÁN todos los datos actuales. Se recomienda exportar una copia de respaldo antes de importar.

Preguntas Frecuentes

▼
🔐 ¿Olvidé mi contraseña? ▼
Contacte al administrador del sistema para restablecer su contraseña. Si es el administrador, puede acceder con las credenciales por defecto o cambiarlas en el panel de Admin Claves.
📊 ¿Los datos se pierden si cierro el navegador? ▼
No. El sistema usa IndexedDB (base de datos local en el navegador). Todos los datos persisten incluso después de cerrar el navegador o apagar la computadora.
🔄 ¿Puedo usar el sistema sin internet? ▼
¡Sí! El sistema funciona 100% offline. No requiere conexión a internet para funcionar después de la primera carga.
📥 ¿Cómo exporto los reportes? ▼
En la sección "Reportes", después de aplicar los filtros deseados, haga clic en "Exportar Reporte". Se descargará un archivo CSV.
👥 ¿Qué pasa si dos gestores tienen el mismo código? ▼
El sistema detecta automáticamente los duplicados y añade un número secuencial (ej: GR4, GR4-1, GR4-2...). Esto garantiza que cada gestor tenga un código único.
💾 ¿Cómo recupero datos si los perdí? ▼
Si realizó exportaciones de copia de seguridad previamente, puede usar el botón "Importar Copia" para restaurar los datos. Se recomienda exportar copias periódicamente.
