⚙️ Configuración de Porcentajes

📌 Información importante:

✓ Los porcentajes se aplican sobre el precio base del producto en USD

✓ El cliente pagará: Precio base + (Precio base × Porcentaje / 100)

✓ Porcentaje 0% significa que se paga el precio exacto sin recargo

✓ Los cambios afectarán SOLO a las ventas futuras, no a las ya registradas
